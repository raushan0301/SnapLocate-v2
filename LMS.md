
👨‍🎓 STUDENT OS
Course & Content Access

Enrolled courses dashboard with progress bars per course
Course materials: lecture notes (PDF/PPT/DOCX), recorded videos, reference links
Week-by-week / module-by-module syllabus view with completion checkboxes
Download materials for offline access
Bookmark specific resources within a course

Attendance

View attendance percentage per subject (critical — most Indian universities have 75% minimum rule)
Color-coded warning: green (safe), yellow (borderline), red (detained risk)
Attendance history: date-wise log of present/absent/late
"How many classes can I miss?" calculator
Detainment alert notifications when attendance drops below threshold

Assignments

View all pending, submitted, and graded assignments in one inbox
File submission (PDF, DOCX, ZIP — max 25MB)
Text/rich-text submission for written answers
Submission deadline countdown timer
Late submission indicator (faculty decides if late is accepted)
Resubmit option (if faculty enables it)
View faculty feedback and rubric score after grading

Quizzes & Tests

Timed online quizzes with auto-submit on timeout
Question types: MCQ, True/False, Short Answer, Fill in the Blank
Randomized question order (anti-cheating)
Results and correct answers shown after submission (faculty-controlled)
Quiz attempt history

Grades & Performance

Gradebook: view marks per component (quiz, assignment, mid-term, end-term)
Current CGPA / SGPA estimate (if grading scale configured)
Grade breakdown per course: internal marks vs external exam marks
Downloadable marks summary

Announcements & Notifications

Per-course announcements from faculty
University-wide notices
Push notifications for: new assignment posted, deadline reminder (24h before), grade released, attendance updated

Discussion Forums

Per-course discussion board
Post questions, reply to threads
Faculty can pin important posts
Upvote helpful answers (Stack Overflow style)

Live Classes / Virtual Sessions

View scheduled live class links (Zoom/Google Meet/MS Teams — just link management, no self-hosting needed for v1)
Recorded session links posted post-class
Class schedule / timetable view

Study Resources

Previous year question papers upload/download (per course)
Syllabus document download
Recommended reading list
